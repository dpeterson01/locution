## Locution 0.1.3

Steadier per-app cleanup and a more reliable model download flow.

## What's new

- **Per-app cleanup follows the focused window** — on macOS, switching windows now reselects the matching cleanup mode as you'd expect, so the right rule applies to the app you're actually typing into.
- **Reliable Ollama model matching** — installed models are matched by their normalized tag, so a model you already have is recognized instead of being offered for download again.
- **Download UI reflects the daemon** — the model download controls now stay in step with whether Ollama is running, rather than prompting when the daemon is unavailable.
- **Clearer Windows install guidance** — the README explains the expected Windows SmartScreen warning on unsigned builds and how to proceed.

## Requirements

- macOS or Windows
- [Ollama](https://ollama.com) running locally for AI cleanup (`qwen3.5:2b` and `gemma4:12b`; Apple Silicon can pull the `-mlx` builds)
