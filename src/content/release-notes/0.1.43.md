## Locution 0.1.43

Sharper cleanup for short dictation and a stronger model for longer thoughts.

## What's new

- **Questions stay questions** — short dictation now keeps a question worded as a question and ends it with a question mark instead of flattening it into a statement. It also keeps every word you said and fixes misheard homophones like "their" and "they're".
- **Clearer split between quick and heavy cleanup** — dictation under 500 characters gets fast, light cleanup on a small local model, while longer dictation routes to a stronger model for heavier polishing.
- **Stronger model for longer dictation** — the heavier cleanup tier now uses Qwen3.5 9B, replacing the previous 8B default. Apple Silicon runs the MLX build.

## Requirements

- macOS or Windows
- [Ollama](https://ollama.com) running locally for AI cleanup (`qwen2.5:3b` for quick cleanup and `qwen3.5:9b`; Apple Silicon can pull the `-mlx` build)
