## Locution 0.1.44

Separate start and stop recording sounds, and a more reliable model for longer cleanup.

## What's new

- **Separate recording sounds** — you can now turn the start-recording sound and the stop-recording sound on or off independently. By default the start sound is on and the stop sound is off. Set them under Settings → Sound.
- **More reliable cleanup for longer dictation** — the heavier cleanup tier now uses Qwen2.5 14B, a non-reasoning model that cleans and returns without stalling. This replaces the previous long-tier model, which could pause for a long time before responding on longer dictation.

## Requirements

- macOS or Windows
- [Ollama](https://ollama.com) running locally for AI cleanup (`qwen2.5:3b` for quick cleanup and `qwen2.5:14b` for longer dictation)

Built on Handy. Thanks to the Handy project, Whisper, and Ollama.
