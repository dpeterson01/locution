## Locution 0.1.4

A guided first-run walkthrough and a smoother Ollama setup.

## What's new

- **Guided dictation walkthrough** — onboarding now ends with a hands-on step that teaches the hold-to-dictate loop and lets you try it live before landing in the app. Reopen it anytime from Settings → General.
- **Smoother Ollama setup** — the guided install reports progress as it runs, so first-time setup is clearer about what's happening.

## Requirements

- macOS or Windows
- [Ollama](https://ollama.com) running locally for AI cleanup (`qwen3.5:2b` and `gemma4:12b`; Apple Silicon can pull the `-mlx` builds)

Built on Handy. Thanks to the Handy project, Whisper, and Ollama.
